# 🖐️ 2025HearAI(2학기)

## 🌟 프로젝트 개요
**"실시간 수어 번역 및 감정 분석 시스템"** 은 청각장애인과 비장애인 간의 원활한 의사소통을 위해 개발하려고 합니다.  
기존 문자 기반 대화 앱과 수어 통역사의 한계를 극복하고, **딥러닝 기반의 실시간 번역 및 감정 분석 기술**을 적용합니다.  
이 저장소는 2025년 2학기 프로젝트의 개발 및 진행 과정을 다룹니다.

✅ **실시간 수어 번역**  <br>
